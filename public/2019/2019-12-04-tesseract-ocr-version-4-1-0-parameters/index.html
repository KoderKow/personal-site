Tesseract OCR Version 4.1.0 Parameters

Written by Kyle Harris with ♥ on 4 December 2019 in R 17 min

Overview

During a project my team was having a hard time tracking down an up to date parameters list for Tesseract OCR. The documentation mentions to use tesseract –print-parameters (two hyphens in front of print) to get the available 600+ options. Personally, I use R at my job so sifting through the terminal and trying to go through a poorly formatted (this may be due to my unfamiliarity to the terminal) list in the terminal can be a headache. I have taken the current parameters list for Tesseract OCR 4.1.0 and made it available on this page and in JSON format in hopes it will help people explore the available options.

Cleaning Process

I used RStudio’s find/replace functionality to clean up the text I copied from the terminal using the tesseract –print-parameters command. The Description column has apostrophe’s removed due to the major conflicts it was causing me during data cleaning. There are some single and double quotes in the Value column, these were intially removed and then re-added. I have triple-checked to ensure they are correct since Value is important when looking at the default value of these parameters.

Thank you Tesseract OCR dev team for this fantastic piece of software and making it open source! :)
